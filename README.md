# SETUP:  
- py -m venv env (py, python, python3, whatevs): buat virtual environment  
- env\Scripts\activate (bisa beda tergantung versi), intinya activate virtual environment di cmd  
- pip install -r requirements.txt

# HOW TO RUN:
- env\Scripts\activate  
- uvicorn main:app --reload
- 127.0.0.1:8000/docs (buat buka Swagger, UI nya FastAPI)
